# Governance Index

## Purpose

This directory holds the authoritative governance docs for scaffolded repositories and the generator contract.

## Read On Demand

- `PROJECT_RULES.md` - scope, precedence, naming, and exception rules
- `ARCHITECTURE_GUARDRAILS.md` - module boundaries and forbidden couplings
- `CONTENT_GOVERNANCE.md` - authored content naming and validation rules
- `QUALITY_GATES.md` - release blockers and exception handling
- `WORKFLOW_RULES.md` - planning, editing, and validation workflow

## Rule

If you need governance detail, read the specific file instead of expanding `AGENTS.md`.
